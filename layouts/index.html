Lantern

An open source Minecraft: Java Edition server implementation.

Open Source

Lantern is completely open source, which means that everyone can contribute. The project can be found on GitHub.

Customizable

Lantern implements the API designed by Sponge. Plugins designed using the API can be used for Lantern.

Compatible

Lantern is compatible with the latest version of Minecraft: Java Edition. You don't need a special client to join the server.

Community

The community is located in a Discord server. This is the place where we talk about development, answer questions, etc. Feel free to join us!
